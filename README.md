# Rock Paper Scissors

Classic hand game implementation in vanilla JavaScript.

## Features
- Real-time score tracking
- Cryptographically secure random computer picks
- Responsive design for all devices
- Accessible UI with ARIA labels

## How to Play
1. Choose rock, paper, or scissors
2. Computer makes a random selection
3. Winner of the round is determined and score updates
4. Click "Reset Game" to start over

## Installation
1. Clone the repository
2. Open `index.html` in your browser
3. Start playing!

## Technologies
- HTML5
- CSS3 (with modern features like CSS Grid, backdrop-filter)
- Vanilla JavaScript (Web Crypto API)

## File Structure
- `index.html` - Game markup and structure
- `style.css` - Styling and responsive design
- `script.js` - Game logic and interactivity

## Game Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Identical choices result in a draw
